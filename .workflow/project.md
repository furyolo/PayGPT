# Project: PayGPT

## What This Is

PayGPT 是一个用于生成 GPT 支付长链接的 Tampermonkey 油猴脚本项目。它面向需要在浏览器中快速生成、复制并使用 GPT 支付链接的使用者，优先把“根据输入信息生成可用长链接”这条路径做稳定。

## Core Value

用户输入必要的支付参数后，脚本必须生成一个格式正确、可复制、可直接打开的 GPT 支付长链接。

## Context

当前仓库为空项目，未检测到现有源码或 manifest。用户给出的原始方向是“项目主要是为 GPT 支付生成一个长链接，即长链接的支付链接，做成一个油猴脚本”。后续补充需求为：用户可选择 coupon 和国家，脚本根据选择调用 ChatGPT checkout 接口生成支付长链接。当前国家映射为美国 `US/USD`、英国 `GB/GBP`、泰国 `TH/THB`。

## Constraints

- **运行环境**：Tampermonkey 用户脚本 — 项目应优先保持单文件或少量静态文件结构，避免引入不必要构建链。
- **架构边界**：纯浏览器侧生成链接 — 不默认新增服务端、数据库或外部托管能力。
- **安全边界**：不处理敏感凭证或绕过式支付能力 — 链接生成逻辑不得扩大到未授权支付行为。
- **依赖策略**：优先使用原生 Web API — 油猴脚本场景下避免擅自引入新依赖。
